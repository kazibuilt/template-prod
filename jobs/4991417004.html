Software Engineering Intern, Device Automation

Internships at Humane in San Francisco, CA

Humane is a team of proven industry experts who have invented, built, and shipped category-defining hardware and software products to billions of people across the globe. We're known for building the audacious, ambitious, and the impossible, and we're doing it again. Our vision for the next shift between humans and computing requires innovation across multiple emerging technologies in hardware and software, from devices to cloud services.

About The Role

As an intern on the Device Automation Software team, you will work with the hardware and software teams to build scalable automated testing to support Humane’s mission to reimagine computing. You will write and automate test plans, build physical and virtual test fixtures, and improve the software test infrastructure by writing tools and bringing infrastructure improvements to existing workflows.

What You Might Do

  • Contribute to Humane’s device automation framework to enable automation of low low level android system code and high level android application code
  • Contribute to existing java/python libraries to create new testing infrastructure
  • Set up dashboards for reporting automated test results
  • Create automated test fixtures for testing hardware devices

Essential Qualifications 

  • Pursuing a Bachelor's or Master’s degree in Computer Science and/or related engineering fields
  • Programming experience in Python, C/C++, and/or Java
  • Experience with Git or other SCM tools
  • Familiarity with Linux, or another Unix like operating system

Preferred Skills

  • Knowledge of one or more continuous integration pipeline software products such as Jenkins, Github Actions, CircleCI, etc
  • Knowledge of Android test frameworks such as the Android ACTS tooling
  • Knowledge of electronics and embedded systems
  • Ability to read schematics

Pay Range: $45-$48/hour
